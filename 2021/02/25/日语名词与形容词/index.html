日语名词与形容词

发表于 2021-02-25 更新于 2020-02-25 分类于 日语

名词

简体形 敬体形
现在 ～ は ～ + だ ～ は ～ + です
现在否定 ～ は ～ + ではない／じゃない ～ は ～ + ではありません／じゃありません
过去 ～ は ～ + だった ～ は ～ + でした
过去否定 ～ は ～ + ではなかった／じゃなかった ～ は ～ + ではありませんでした／じゃありませんでした

形容词

形容词简体形式

简体形 い形容词 な形容词
现在 忙しい 便利だ
现在否定 忙しくない 便利ではない／便利じゃない
过去 忙しかった 便利だった
过去否定 忙しくなかった 便利ではなかった／便利じゃなかった

形容词敬体形式

敬体形 い形容词 な形容词
现在 忙しいです 便利です
现在否定 忙しくないです／忙しくありません 便利ではないです／便利ではありません
过去 忙しかったです 便利でした
过去否定 忙しくなかったです／忙しくありませんでした 便利ではなかったです／便利ではありませんでした

特殊形容词[いい], 活用变形按照[よい]来变化

简体形 敬体形
现在 いいです
现在否定 よくない よくないです／よくありません
过去 よかった よかったです
过去否定 よくなかった よくなかったです／よくありませんでした

形容词接续

い形容词 な形容词
て形 い形容词干 + く + て
表示并列或递进的关系, 或者表示原因
牛丼は安くて、美味しいです。		 な形容词干 + で
表示并列或递进的关系, 或者表示原因
この辺りはとても静かで、風景もきれいです。
+ 名词 い形容词基本形 + 名词
い形容词词干 + かった + 名词		 な形容词词干 + な + 名词
な形容词词干 + だった + 名词
+ 动词 い形容词词干 + く + 动词
牛肉を薄く切ります		 な形容词词干 + に + 动词
