新编日本教程复习之3

发表于 2021-03-22 更新于 2020-03-22 分类于 日语

新编日语教程. 第9至第12课复习.

1. 名1は + 名2より + イ形/ナ形です

表示比较, 意思是"名词1比名词2更加…". 助词"より"表示比较的基准和对象. 后面的"イ形/ナ形"要用肯定的表达方式.

  • 地下鉄はバスより速いです
  • 東京は京都より賑やかです
  • 去年の夏は今年の夏より暑かったです

2. 名2は + 名1ほど + イ形/ナ形

名2は + 名1ほど + イ形くないです/イ形くありません

名2は + 名1ほど + ナ形ではないです/ナ形ではありません

表示比较, 意思是"名词2不如名词1…", 后面要用否定的表达方式. 助词"ほど"表示比较的基准.

  • 今年の冬は去年の冬ほど寒くないです
  • 大阪は奈良ほど静かではありません
  • 昨日の映画は今日の映画ほど面白くなかったです

3. 名1より + 名2のほうが + イ形/ナ形です

表示比较, 意思是"比起名词1, 名词2更加…", 助词"より"提示比较的对象

  • 田中さんより佐藤さんのほうが若いです
  • 秋より春のほうが好きです
  • ダニエルさんは木村さんよりスピートは早いですが、サープはダニエルさんより木村さんのほうが上手ですね

最后一个例句中, "スピートは、サープは~"的两个"は"是表示对比和强调.

4. 名1と + 名2とどちらが イ形/ナ形ですか

表示比较的疑问句形式, 意思是"名词1和名词2哪一个更加…", 此句型只能用来比较两个事物. 其回答可以用"~のほうが ~です(…更…)"或者"どちらも ~です(两个都…)".

  • A: 日本語の英語とどちらが難しいですか
  • B: 英語より日本語のほうが難しいです
  • A: 富士山と高尾山とどちらが高いですか
  • B: 富士山のほうが高いです

5. 名1は + 名2で、名3です

在名词作谓语的句型中, 可以用"名 + で"的形式连接两个并列的名词

  • 鈴木さんは王さんの友達で、大学の三年生です
  • あそこの白いビルは新しい建物で、図書館です

6. が(提示主语)

助词"が"提示句子的主语, 强调"が"前面的内容. "が"前面的内容多为之前没有提及的新内容/新信息等

  • 私が担当者です(强调负责人是我, 而不是其他人)
  • ダニエルさんが勝ちました(强调获胜者是丹尼尔, 而不是其他人)

如果将上述例句中的"が"变成"は"的话, 句子的意思就会发生变化, 如:

  • 私は担当者です

这句话说明我的身份是负责人, 而不是其他职务, 只是普通的陈述.

7. 名/ナ形 + に + なります

"なります"表示性质或状态的变化, 意思是"变得…/成为…", 其基本形式是"なる". "に"表示变化的结果. 名词可以直接接"に なります", ナ形容词接"なります"时, 要去掉词尾的"だ", 再加上"に なります"

分类 词例 + に なります
名词 医者 医者になります
ナ形容词 元気だ 元気になります
  • 娘は医者になります
  • 彼は以前より真面目になりました

8. イ形 + く + なります

"なります"表示性质和状态的变化, 意思是"变得…/成为…", イ形容词连接"なります"时, 要将词尾假名"い"变成"く"后, 再加上"なります"

分类 词例 + く なります
イ形容词 高い 高くなります
イ形容词 安い 安くなります
  • 物価は高くなりました
  • だんだん暖かくなります
  • うちのワンちゃんは大きくなりました

9. 名/ナ形 + に + します

该句型表示因主语的意志性行为引起事物发生变化, 意思是"把…变得…/使…成为…", "に"表示结果, "する"强调主语的意志性行为. 名词可以直接接"にします", ナ形容词与"にします"连接时, 要去掉词尾的"だ", 再加上"に します"

分类 词例 + に します
名词 禁煙 禁煙にします
ナ形容词 綺麗だ 綺麗にします
  • 会議室を禁煙にします
  • 部屋を綺麗にします

此外, 在购物/点餐时, 用"名 + に + します"可以表示从几个选项中选定一项, 或者向对方表明自己的决定.

  • 田中さんは何にいますか
  • 私はＡセットにします

10. イ形 + く + します

该句型表示因主语的意志性行为引起事物发生变化, 意思是"把…变得…/使…成为…". イ形容词与"します"连接时, 要将词尾假名"い"变成"く", 再加上"します".

分类 词例 + く します
イ形容词 安い 安くします
イ形容词 高い 高くします
  • テレビの音を大きくします
  • 暑いですから、髪を短くしました

11. 名1で + 名2が + 一番 ~です

表示在某个范围内, 名词2最具有某种性质, 意思是"在…中, 名词2最…", 助词"で"表示范围.

  • 果物の中でスイカが一番好きです
  • A: クラスメートの中で李さんが一番若いですか
  • B: はい、そうです

12. 名で + 疑问词が + 一番 ~ですか

该句型中用于询问三个或三个以上的事物中哪一个最具有某种性质, 或者询问在某个范围内, 哪一个最具有某种性质, 意思是"在…之中, 哪一个…最…?".

  • A: スポーツの中で何が一番得意ですか
  • B: テニスが一番得意です
  • A: 四季の中にどの季節が一番好きですか
  • B: 春が一番好きです

此外也可以用"名1と 名2と 名3の中で、どれが 一番 ~ですか"来提问.

  • A: イチゴとりんごとバナナの中で、どれが一番好きですか
  • B: イチゴが一番好きです

13. ~は ~が ~です

该句型中, 助词"は"提示句子的主体, 其谓语由一个主谓结构构成. 助词"~が"提示谓语部分的主语.

  • 象は鼻が長いです("象"是句子的主体, "鼻"是谓语部分的主语)
  • この電子辞書は軽量、薄型が特徴です("電子辞書"是句子的主体, "軽量、薄型"是谓语部分的主语)

14. 动词て形

动词て形是动词的一种变化形式, 主要用于连接前后句, 可以表示并列/时间上的先后顺序/因果关系等. 动词て形也可以作为一种接续的形式在句型中使用. 动词て形是最常见/最基本的动词变化形式之一.

一类动词变成て形有三种音变形式

以"う/つ/る"结尾的动词变成て形时, 发生"促音变", 要将词尾的"う/つ/る"变成"って"

以"ぶ/む/ぬ"结尾的动词变成て形时, 发生"拨音变", 要将词尾的""变成"んで"

以"く/ぐ"结尾的动词变成て形时, 发生"い音变", 要将词尾的"く/ぐ"分别变成"いて/いで", 但是"行く"的て形为"行って", 需要特殊记忆.

以"す"结尾的动词变成て形时, 不发生音变, 将词尾的"す"变成"して"即可.

二类动词变成て形时, 不发生音变, 将词尾的"る"变成"て"即可.

三类动词"する"和"来る"的て形分别是"して"和"来て". "来る"变成て形后读音发生变化, 要读作"きて".

分类 词例 音变类型 て形
一类动词 買う 促音变(~って) 買って
持つ 促音变(~って) 持って
帰る 促音变(~って) 帰って
呼ぶ 拨音变(~んで) 呼んで
読む 拨音变(~んで) 読んで
死ぬ 拨音变(~んで) 死んで
書く い音变(いて/いで) 書いて
泳ぐ い音变(いて/いで) 泳いで
話す 无音变 話して
二类动词 食べる 无音变 食べて
起きる 无音变 起きて
開ける 无音变 開けて
三类动词 する 特殊音变 して
来る 特殊音变 来て

15. 动て、动

动词"て形"可以连接两个或两个以上的动词, 表示动作按照先后顺序发生, 意思是"…之后".

  • 朝ご飯を食べて、学校へ行きます
  • 昨日図書館へ行って、本を借りました
  • スーパーへ行って、買い物をして、家へ帰ります

16. 动て から

"动て形 + から"表示两个动作按照时间先手顺序相继发生, 意思是"在…之后".

  • 朝ご飯を食べてから。出かけます
  • 会社に行ってから、電話をします
  • 宿題をしてから、テレビを見ます

17. 动て います

"动て形 + います"可以表示正在进行的动作或者经常, 反复进行的动作.

  • 鈴木さんは今新聞を読んでいます
  • 週に二回日本語教室に通っています

"动て形 + います"还可以表示状态的持续以及动作结果的存续.

  • 彼女は上海に住んでいます
  • 私がドアを開けました。今ドアは開いています

18. 动て ください

"动て形 + ください"表示指令或者请求, 多在请求某人做某事时使用, 意思是"请…".

  • 暑いですから、窓を開けてください
  • ここはお名前とご住所を書いてください
  • しみません、今忙しいですから、ちょっと待ってください

19. 动て みます

"动て形 + みます"表示尝试着做某事, 意思是"试着做…"

  • 牛丼を作ってみます
  • 新しい服を試着してみます
  • A: これは新発売のスポーツドリンクですよ
  • B: じゃ、ちょっと飲んでみますね

20. イ形/ナ形 + 动

形容词可以通过变形修饰动词, 是类似于副词的用法. イ形容词修饰动词时, 要将词尾的假名"い"变成"く". "いい"是"よい"的口语形式. "いい/よい"修饰动词时应该变成"よく". ナ形容词修饰动词时, 要将词尾的"だ"变成"に".

分类 词例 副词化 与动词连用
イ形容词 早い 早い + く 早く起きます
ナ形容词 綺麗だ 綺麗だ + に 綺麗に化粧します
  • 車を安く売りました
  • 学生たちは静かに座っています
  • A: 佐藤さん、昨日の宿題はよくできましたよ
  • B: ありがとうございます

21. 动ては いけません

"动て形 + は + いけません"表示禁止, 意思是"不能…". 此句型多用来说明规章制度等, 一般不对长辈/上级等使用.

  • ここに車を止めてはいけません
  • 美術館で写真を撮ってはいけません
  • 廊下で大きな声で騒いではいけません

22. 动ても いいです

"动て形 + も + いいです"表示许可, 意思是"可以…/…也行"

  • 今から行ってもいいです
  • 少し休んでもいいですよ
  • A: 荷物をここに置いてもいいですか
  • B: ええ、いいですよ
  • A: 教室が熱くなりましたから、窓を開けてもいいですか
  • B: ええ、いいですよ

23. 动て しまいます

"动て形 + しまいます"表示动作完成和结束, 在句子中常以过去时出现, 带有遗憾/后悔的语气或者表示彻底做完某事.

  • 財布を無くしてしまいました
  • 愛犬が死んでしまいました
  • ケーキを全部食べてしまいました

24. 名までに

"までに"前面多与表示时间/期限的名词连用, 意思是"在…之前/到…为止". 后项的内容多是某个动作或者某件事情, 表示在前项的戒指日期之前, 完成后项的动作或事情.

  • 明日は朝九時までに来てください
  • 今週の金曜日までにレポートを出してください

此外, "までに"表示在截止日期之前的任何一个时间点, 完成后项的动作或事情即可. 所以"までに"后面的动词一般不用表示持续性意义的动词.

  • 来週の金曜日までに待ってください (x)
  • 来週の金曜日までに提出してください (o)

"待っ"是一个包含持续性意义的动词, 不能与"までに"连用, 上述错误例句中的"までに"要改成"まで", 表示动作的持续.

25. ので

"ので"表示原因/理由, 意思是"因为…". "ので"前面多叙述客观事实, 不用命令/祈使等表达方式. "ので"前面可以与动词基本形或者动词形连接, 也可以与"イ形"/"ナ形 + な"/"名词 + な"等形式连接

分类 词例 常见连接方式 + ので
动词 行く 行く/行きます 行くので
行きますので
形容词 高い 高い 高いので
形容词 便利だ 便利だ + な 便利なので
名词 学生 学生 + な 学生なので
  • 用事がありますので、お先に失礼します
  • このかばんは品質がいいので、買いました
  • 今日は日曜日なので、人がたくさんいます
  • この通販サイトはとても便利なので、よく利用します
