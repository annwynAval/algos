新编日本教程复习之4

发表于 2021-03-27 更新于 2020-03-27 分类于 日语

新编日语教程. 第13至第16课复习.

1. 动ない形

动词ない形式动词的一种变化形式, 表示否定, 也可以作为一种接续的形式在句型中使用. 动词ない形的变化规则如下:

一类动词变成"ない形"时, 将词尾的"う"段假名变成相应的"あ"段假名, 再加上"ない". 要特别注意以"う"结尾的动词, 要把"う"变成"わ"再加上"ない".

二类动词变成"ない形"时, 去掉词尾的"る"再加上"ない".

三类动词"する"变成"しない", "来る"变成"来ない".

另外, 动词"ある"的"ない形"是"ない".

分类 词例 ない形
一类动词 買う 買わない
行く 行かない
話す 話さない
待つ 待たない
死ぬ 死なない
呼ぶ 呼ばない
読む 読まない
帰る 帰らない
二类动词 起きる 起きない
食べる 食べない
三类动词 勉強する 勉強しない
来る 来ない

2. 动ないでください

"~ないでください"表示否定的请求或命令, 意思是"请不要…", 动词与"ないでください"连接时, 要用"动ない形 + でください"的形式. 该句型一般不对长辈或上级使用.

  • 私は大丈夫です。心配しないでください。
  • パスポートなどは忘れないでくださいね。
  • ここは通路ですから、荷物などを置かないでください

3. 动なくてもいいです

"~なくてもいいです"表示不用做某事也可以, 意思是"不用…也行". 动词与"~なくてもいいです"连接时, 要将动词"ない形"中的"ない"去掉, 再加上"なくてもいいです". 其疑问句形式"~なくてもいいですか"用来征求对方的许可或意见, 意思是"不用…也可以吗?"

词例 ない形 连接方式 + なくてもいいです
行く 行かない 行かない 行かなくてもいいです
食べる 食べない 食べない 食べなくても
残業する 残業しない 残業しない 残業しなくても
来る 来ない 来ない 来なくても
  • 今日はあまり忙しくありませんから、残業しなくてもいいです
  • 明日は土曜日ですから、朝早く起きなくてもいいです
  • この件は部長に報告しなくてもいいです
  • 詳しいことを調べてから、報告しましょう

4. 动なければなりません

"动なければなりません"表示不得不做某事, 意思是"必须要…". 动词与"なければなりません"连接时, 要将"ない形"中的"ない"去掉, 再加上"なければなりません".

词例 ない形 连接形式 + なければなりません
行く 行かない 行かない 行かなければなりません
食べる 食べない 食べない 食べなければなりません
残業する 残業しない 残業しない 残業しなければなりません
来る 来ない 来ない 来なければなりません
  • 金曜日までに宿題を提出しなければなりません
  • 今日中に家賃を払わなければなりません
  • 急用ができて、すぐ家に帰らなければなりません

5. イ形/ナ形 + て(原因)

イ形容词/ナ形容词的"て形"可以用来表示前后两项的并列关系外, 还可以表示前后两项的因果关系.

  • あの店の料理は美味しくて、いつも長い行列ができています
  • この町はとても有名で、観光客が多いです

6. 动/イ形/ナ形 + 過ぎる

"~過ぎる"表示超出一定的限度, 意思是"过度/过于…". 与"過ぎる"连接时, 动词要用"ます形"去掉"ます"后的形式, イ形/ナ形要去掉词尾, 用词干与"過ぎる"连接, 动词/イ形/ナ形与"過ぎる"连接后会变成一个复合词, 作为一个二类动词来使用, 词尾可以活用.

分类 词例 连接形式 + 過ぎる
动词 食べる 食べます 食べ過ぎる
イ形容词 高い 高い 高過ぎる
ナ形容词 静かだ 静かだ 静か過ぎる
  • 夕べ、お酒を飲み過ぎるました
  • その靴は綺麗ですが、値段が高過ぎて、買いませんでした
  • 自習室は静か過ぎて、眠くなりました

此外, "過ぎる"还可以变成"~過ぎ"的形式. 此时作为接尾词, 与前面的词复合成一个名词, 如: "飲み過ぎ/言い過ぎ/食べ過ぎ"等.

  • 飲み過ぎは体によくないです

7. 动 + ながら

"~ながら"表示两个动作同时进行, 意思是"一边…一边…". 表示两个动作同时进行时, 后项是主要的动作, 前项的动作用来描述后项动作进行时的状态. 动词与"ながら"连接时, 要用"ます形"去掉"ます"后的形式.

分类 ます型 连接形式 + ながら
話す 話します 話します 話しながら
食べる 食べます 食べます 食べながら
する します します しながら
  • 音楽を聞きながら、宿題をやります
  • メモを取りながら、授業を聞いています
  • 友達とおしゃべりをしながら、食事をしています

8. 动た形

动词"た形"是动词的一种变化形式, 用来叙述过去的动作/状态, 也可以作为一种接续的形式在句中使用. 动词"た形"的变化规则与动词"て形"的变化规则相同. 动词"行く"的"た形"为"行って", 作为特例记忆.

分类 基本形 て形 た形
一类动词 買う 買って 買った
持つ 持って 持った
帰る 帰って 帰った
呼ぶ 呼んで 呼んだ
読む 読んで 読んだ
死ぬ 死んで 死んだ
書く 書いて 書いた
泳ぐ 泳いで 泳いだ
話す 話して 話した
二类动词 食べる 食べて 食べた
起きる 起きて 起きた
開ける 開けて 開けた
三类动词 する して した
来る 来て 来た

9. 动/イ形/ナ形/名词 + ほうがいいです

"~ほうがいいです"表示建议, 意思是"还是…为好". 与"~ほうがいいです"连接时, 动词要用"た形"或者"ない形", イ形容词可以直接连接. ナ形容词要用"ナ形词干 + な + ほうがいいです"的形式. 名词要用"名 + の + ほうがいいです"的形式.

分类 词例 连接形式 + ほうがいいです
动词 行く 行った/行かない 行ったほうがいいです行かないほうがいいです
形容词 安い 安い 安いほうがいいです
形容词 静かだ 静かだ + な 静かなほうがいいです
名词 コーラ コーラ + の コーラのほうがいいです
  • 事前にホテルを予約したほうがいいです
  • 大雨が降っていますので、今日は出かけないほうがいいです
  • 旅行の時、荷物は軽いほうがいいです
  • 商品の品質はＡ社よりＢ社のほうがいいです

10. 动/名 + 後で

"~後で"表示时间的先后顺序, 意思是"在…之后", 多指做完前项的事情之后, 再做后项的事情. 与"~後で"连接时, 动词要用"た形", 名词要用"名 + の後で"的形式.

  • 準備体操をした後で、水泳を始めます
  • 朝ご飯を食べた後で、学校へ行きます
  • 仕事の後で、コンサートに行きます

11. 敬体形和简体形

日语有敬体和简体之分. 敬体是以"です/ます"出现的形式, 一般在与长辈/上级的对话中使用. 简体主要用于关系亲密的朋友/晚辈/下级使用.

动词的简体形
敬体形 简体形
现在肯定 行きます 行く(基本形)
现在否定 行きません 行かない(ない形)
过去肯定 行きました 行った(た形)
过去否定 行きませんでした 行かなかった

动词的简体主要有"动词基本形/动词ない形/动词た形/动词なかった"的形式.

动词"ある"的简体形变化比较特殊.

敬体形 あります ありません ありました ありませんでした
简体形 ある ない あった なかった
  • 王さんは来週日本に行きます
    王さんは来週日本に行く
  • 朝ご飯はもう食べました
    朝ご飯はもう食べた
  • お酒は飲みません
    お酒は飲まない
  • 彼女に会いませんでした
    彼女に会わなかった
イ形容词的简体形
敬体形 简体形
现在肯定 忙しいです 忙しい
现在否定 忙しくないです忙しくありません 忙しくない
过去肯定 忙しかったです 忙しかった
过去否定 忙しくなかったです忙しくありませんでした 忙しくなかった
  • このかばんは非常に安いです
    このかばんは非常に安い
  • 図書館はあまり遠くありません
    図書館はあまり遠くない
  • 昨日の映画は面白かったです
    昨日の映画は面白かった
  • 先週はあまり忙しくありませんでした
    先週はあまり忙しくなかった
ナ形容词的简体形
敬体形 简体形
现在肯定 有名です 有名だ
现在否定 有名ではないです有名ではありません 有名ではない
过去肯定 有名でした 有名だった
过去否定 有名ではなかったです有名ではありませんでした 有名ではなかった
  • その先生はとても親切ですよ
    その先生はとても親切だよ
  • この機械の操作は簡単ではありません
    この機械の操作は簡単ではない
  • 昔、この辺りには地下鉄がなくてとても不便でした
    昔、この辺りには地下鉄がなくてとても不便だった
  • 二十年前にここはあまり賑やかではありませんでした
    二十年前にここはあまり賑やかではなかった
名词的简体形
敬体形 简体形
现在肯定 学生です 学生だ
现在否定 学生ではないです学生ではありません 学生ではない
过去肯定 学生でした 学生だった
过去否定 学生ではなかったです学生ではありませんでした 学生ではなかった
  • 鈴木さんは大学の三年生です
    鈴木さんは大学の三年生だ
  • 彼は学生ではありません
    彼は学生ではない
  • 彼女は看護婦でした
    彼女は看護婦だった
  • ここは昔学校ではありませんでした
    ここは昔学校ではなかった
简体形的疑问句以及回答

简体形的疑问句是有简体句加上问号构成的, 句末读升调. 简体形疑问句的肯定回答是"うん", 否定回答是"ううん". 在简体会话中, 常用"うん"代替"はい", 用"ううん"代替"いいえ". 简体形的疑问句以及回答常用在关系亲密的朋友之间.

  • A: 佐藤君、明日学校に行く？
    B: ううん、行かない
  • A: 一緒に映画を見に行かない？
    B: うん、いいね
  • A: 昨日のコンサート、行った？
    B: ううん、行かなかったよ

12. 简体形 + と思います

"简体形 + と思います"用来描述说话人的观点或者说话人所思考的内容, "~と思います"前面一定要用简体形, 意思是"我想…/我认为…/我觉得…".

  • セールは来週から始まると思います
  • 彼女はたぶん花見に行かないと思います
  • 王さんはもう日本に行ったと思います

14. 动 + たいです

"~たいです"表示个人的愿望, 意思是"想…". 其否定形式是"~たくないです"或"~たくありません", 过去式是"~たかったです". 过去否定形似是"~たくなかったです". "~たいです"多用来表示第一人称的愿望, 或者在疑问句中使用. 但不能直接表示第三人称的愿望. 动词接"~たいです"时, 要用"ます形"去掉"ます"后的形式.

词例 ます形 连接方式 + たいです
行く 行きます 行きます 行きたいです
食べる 食べます 食べます 食べたいです
  • フランスへ旅行に行きたいです
  • 食欲がなくて、何も食べたくないです
  • 彼と一緒に映画を見に行きたかったです
  • 昨日は一日中何もしたくなかったです

"たい"的对象语用"を"或者"が"表示. 用"を"的时候, 句子的重点在谓语上. 用"が"时, 句子的重点在对象语上.

  • バナナを食べたいです. => 强调"食べる"
  • バナナを食べたいです. => 强调"バナナ"

15. 名 + が欲しいです

"名 + が欲しいです"表示个人的愿望. 意思是"想要…". 表示的是第一人称的愿望, 一般不用于第三人称. 其疑问句形式是"何が欲しいですか"." ~欲しいです"的否定形式是"~欲しくないです"或者"~欲しくありません". 过去式是"~欲しかったです", 过去否定形式是"~欲しくなかったです".

  • 新しいかばんが欲しいです
  • クリスマスのプレゼントは何が欲しいですか
  • わたしは何も欲しくないです
  • 誕生日のプレゼントは新しいゲーム機が欲しかったです

16. 动意向性

动词的意志性是动词的一种变化形式, 用来表示说话人的决心/意志. 也可以作为一种接续的形式在句中使用.

一类动词变成意志形时, 将"う"段假名变成"お"段假名, 再加上"う"

二段动词变成意志形时, 去掉词尾的"る", 再加上"よう"

三类动词"する"的意志形是"しよう", "来る"的意志形是"来よう"

分类 基本形 意志形
一类动词 買う 買おう
行く 行こう
泳ぐ 泳ごう
話す 話そう
待つ 待とう
死ぬ 死のう
遊ぶ 遊ぼう
読む 読もう
帰る 帰ろう
二类动词 食べる 食べよう
三类动词 する しよう
来る 来よう

17. と(共同做动作的同伴或行为的对象)

"と"除了连接两个名词之外, 还可以表示一起完成某个动作/行为的同伴/对象, 意思是"和…/同…".　

  • 皆と相談してから、決めます
  • 佐藤さんと一緒に旅行に行きました
  • 小野さんはジョンさんと結婚しました

18. 名1か 名2(选择)

"か"可以表示选择, 意思是"或者".

  • 月曜日か火曜日に図書館に行きたいです
  • 冬休みに北海道かハワイへ旅行に行くます
  • バナナかりんごを買ってください

19. 自动词和他动词

日语中的动词可以分为自动词和他动词. 自动词作谓语时, 不带宾语, 即动作/行为不涉及其他对象. 他动词作谓语时, 要求有宾语, 即动作/行为要涉及其他对象. 动作/行为的对象用助词を表示

自动词 例句 他动词 例句
開く ドアが開く 開ける ドアを開ける
動く 木の枝が動く 動かす 車を動かす
落ちる りんごが落ちる 落とす ペンを落とす
消える 電気が消える 消す 電気を消す
壊れる パソコンが壊れる 壊す 玩具を壊す
閉まる ドアが閉まる 閉める ドアを閉める
つく 電気がつく つける 電気をつける
止まる 車が止まる 止める 車を止める
出る 涙が出る 出す 手紙を出す
入る 車が車庫に入る 入れる お茶を入れる
始まる 授業が始まる 始める 仕事を始める
割れる コップが割れる 割る スイカを割る
  • 部屋の電気がつきました
  • 電車のドアが開きました
  • コップが割れましたね
    ええ、うっかりして割りました
  • 王さんがドアを閉めましたか
    いいえ、自動的に閉まりました

20. 他动词てあります

"他动词てあります"表示动作结果的存续, 指有意进行某动作之后, 其结果还存续着. 动作作用的对象是该句型描述的重点.

有意进行的动作 结果的存续
私は地図をかばんに入れました 地図はかばんに入れてあります
王さんが壁にカレンダーをかけました 壁にカレンダーがかけてあります
母が部屋にクリスマスツリーを飾りました 部屋にクリスマスツリーが飾ってあります

21. 动 + 名(动词简体形修饰名词)

动词简体形或者以动词简体形结束的短语/句子, 可以作定语修饰名词

词例 简体形 + 名词 意思
料理を作ります 料理を作る時間 做饭的时间
今使いません 今使わない資料 现在不用的资料
王さんが留学しました 王さんが留学した大学 小王留学过的大学
誰にも話しませんでした 誰にも話さなかったこと 没有跟任何人说过的事情
  • 忙して、仕事以外のことを考える余裕がないです
  • これはあの有名な作家が書いた本です

22. 动 + たり/ 动 + たりします

"~たり/ ~たりします"用来列举若干有代表性的动作, 动词与"~たり/~たりします"连接时, 要将动词"た形"中的"た"变成"たり".

  • 休みの日は部屋の掃除をしたり、買い物したりします
  • 桜の木の下で、お酒を飲んだり、友達とおしゃべりをしたります
  • 日曜日はよくドライブに行ったり、映画を見たりします

23. 动/名 + まま

"~まま"表示状态的保持, 意思是"在…状态下, 就…". 与"~まま"连接时, 动词要用"た形"或者"ない形". 名词要用"名 + のまま"的形式

  • エアコンをつけたまま、出かけました
  • 昨日は夕飯を食べないまま、寝てしまいました
  • あの本は買ったまま、まだ読んでいません

24. 名 + がします

"~がします"常与表示声音/气味/味道/感觉等词一起使用, 用来描述听觉/味觉/嗅觉/感觉等.

  • この飲み物はレモンの味がします
  • 隣の部屋からピアノの音がします
  • このシャンプーは　バラの香りがします

25. 动て来ます/て行きます(动作的方向)

"动て形 + 来ます"可以表示动作移动的方向由远及近, 意思是"…来".

  • 地下鉄のはたくさんの人が乗ってきました
  • 家からお弁当を持ってきました

"动て形 + 行きます"可以表示动作移动的方向由近及远, 意思是"…去".

  • 李さんはゴールへ向かって走っていきました
  • 鳥が遠くへ飛んでいきました
